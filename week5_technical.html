Ruby Classes

Let's define a class. A class is the blueprint of a program that organizes methods and variables. To create a class in Ruby, start with the word class followed by the class name, capitalized. Like this:

class Person
end

We use classes whenever we want to organize data about objects that we want to differentiate between later on by creating a new instance of the class. So, we can create variables associated with the object as follows:

class Typical_person
num_of_eyes = 2
num_of_noses = 1
num_of ears = 2

We can update information about our object with methods. Like this:

class Typical_person
num_of_eyes = 2
num_of_noses = 1
num_of ears = 2
def name(person_name)
puts person_name.capitalize
end

Classes are extremely important for organizing data.
